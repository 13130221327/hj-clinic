# 何江牙所患者记录系统

一个轻量的 Web 应用，用于记录每日患者就诊信息和费用，支持手机和 PC 访问，部署简单（无需安装独立数据库、无需额外依赖包）。

## 功能

- 新增患者就诊记录（日期、姓名、电话、项目、费用、支付方式、备注）
- 记录列表展示 + 条件筛选（按日期/姓名）
- 一键删除记录
- 简单统计：今日、本月、累计的人次和费用
- 响应式页面，手机和电脑都可操作

## 技术方案

- Python 标准库 `http.server` 实现 Web 服务
- 数据存储：本地 JSON 文件（`data/records.json`）
- 前端样式：Bootstrap 5 CDN

## 快速启动

```bash
python app.py
```

打开：<http://localhost:8000>

## 部署建议（易部署）

### 方式 1：直接运行（内网电脑/小主机）

```bash
python app.py
```

### 方式 2：做成系统服务（Linux）

可通过 `systemd` 将 `python /path/to/app.py` 配置为后台服务，实现开机自启。

> 数据默认保存在 `data/records.json`，备份该文件即可保留历史记录。
